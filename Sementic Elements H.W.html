1.What is Programming?

Ans: Programming is the process of writing instructions for a computer to perform a task. Programmers use a variety of languages to write code, which is a set of step-by-step commands that tell the computer what to do
Programming is the technological process of writing sequences of instructions, known as programs, that tell a computer how to perform specific tasks and solve problems. It involves both the conceptualization of these instructions and the coding to implement them.
